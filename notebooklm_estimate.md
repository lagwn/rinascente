# 概算見積り案

## 1. 見積りの前提
- 本見積りは、公開サイト制作に加えて `CMS / 管理画面` と `YUMEHO / MICA30 共通の会員機能` までを含めた概算です。
- 対象範囲は `Rinascentes コーポレートサイト 11ページ`、`YUMEHO 12ページ`、`MICA30 9ページ` の計32ページに加え、共通会員認証5ページと管理画面です。
- 既存プロトタイプの構成と既存実装を活かし、共通テンプレートと共通パーツを使って制作する前提です。
- 原稿、写真、製品情報は支給または既存素材を活用する前提です。
- 表示最適化は PC / スマートフォン対応を含みます。
- 金額は `税別` の概算です。
- 会員機能は `共通ログイン`、`設置方法 / 利用方法動画`、`購入履歴` を基本範囲とします。
- 会員認証エリアは Rinascentes 管理下に配置します。
- 購入履歴は管理画面から登録・更新する前提とし、オンライン決済連携は本体範囲に含めません。
- デザインは既存プロトタイプのブラッシュアップを前提とし、全面リニューアルは含めません。

## 2. 概算見積り内訳

| 項目 | 内容 | 金額 |
| --- | --- | ---: |
| 企画・要件整理 | ヒアリング、要件整理、全体方針の確定 | 40,000円 |
| 情報設計・画面整理 | サイトマップ整理、機能整理、導線整理 | 50,000円 |
| デザイン調整・共通UI設計 | 既存プロトタイプのブラッシュアップ、共通ルール整理 | 60,000円 |
| 公開サイト制作 | Rinascentes + YUMEHO + MICA30 の公開ページ制作（32ページ） | 260,000円 |
| CMS / 管理画面導入 | FAQ、一部掲載情報、動画、購入履歴の更新導線整備 | 90,000円 |
| 共通会員機能実装 | ログイン、再設定、会員トップ、動画閲覧、購入履歴閲覧（Rinascentes管理下） | 80,000円 |
| テスト・公開調整 | 表示確認、導線確認、公開前最終調整 | 20,000円 |
| **合計** |  | **600,000円** |

## 3. この見積りで実現できる内容
- 公開に必要な主要ページ一式の整備（3サイト計32ページ + 会員認証5ページ）
- Rinascentes コーポレートサイトによる企業信頼の形成（ブランドメッセージ、経営チーム、沿革、プレスリリース）
- YUMEHO の技術優位性訴求（天井走行式デュアルレールの比較セクション、動画キャプチャ画像）
- YUMEHO の導入シミュレーション（ハーネス追加数選択対応）
- MICA30 の製品訴求強化（課題提起セクション、カウンターストリップ）
- 各サイト間を行き来しやすいサイトスイッチャー導線
- デバイスごとの表示最適化
- CMS / 管理画面の最低限の導入
- 両製品共通で利用できる会員機能の基本実装
- 会員向け動画と購入履歴の閲覧導線の整備
- 既存プロトタイプをベースにした効率的な制作

## 4. 見積りに含まないもの
- 大幅な原稿制作や取材ライティング
- 写真撮影、動画撮影、図版制作
- 複数権限ロールを持つ高機能CMS
- 会員ごとの細かな権限分岐やマイページ拡張
- スレッド形式のコミュニティ機能
- YUMEHO / MICA30 両製品からのオンライン決済機能
- 決済に伴う購入履歴の自動反映
- 多言語化
- 広告運用やSEO記事制作

## 5. 本体見積りの考え方
- 公開ページ数は32ページと多いため、完全新規ではなくテンプレート展開を前提とします。
- 会員機能は両製品共通化し、Rinascentes 管理下に配置して管理と運用をシンプルにします。
- 購入履歴はオンライン決済と切り離し、管理画面から手動登録できる構成にすることで、本体予算内に収めます。

## 6. スケジュールの目安
- 1週目: 要件確定、画面整理、デザイン調整方針の確定
- 2週目: 公開ページ制作、CMS / 管理画面、共通会員機能の実装
- 3週目: 結合テスト、修正、公開準備
- 全体目安: `約3週間`

## 7. 本体見積りの前提条件
- 既存プロトタイプのHTML、画面構成、既存パーツを最大限流用する
- ページごとに完全新規デザインを行わず、共通テンプレートを横展開する
- CMS は簡易管理方式とし、更新範囲を限定する
- 会員機能は基本機能に絞り、複雑な権限設計を行わない
- 購入履歴は手動更新を前提とし、自動連携は行わない
- スレッド機能と決済機能は本体から切り分ける
- 文章や素材は支給ベースで進め、制作工数を抑える

## 8. 提案まとめ
- 本体予算 `600,000円` の想定で、公開サイトに加えて `CMS / 管理画面`、`両製品共通の会員機能` まで含めた初期構築が可能です。
- ただし成立条件は、`既存プロトタイプ流用`、`機能範囲の絞り込み`、`3週間での迅速な確認進行` です。
- `オンライン決済` と `スレッド形式のコミュニティ機能` は、別見積りのオプションとして段階追加する進め方が安全です。
