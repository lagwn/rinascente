## 1. 前提整理

### YUMEHO の価値提案（既存）

| 訴求ポイント | 内容 |
|-------------|------|
| 人件費削減 | 見守り1名で運用可能（従来2-3名） → 年間約254万円削減 |
| 訓練機会増加 | 同スタッフ数で1.5倍の訓練提供 |
| 転倒事故リスク低減 | ハーネス + デュアルレールで物理的に防止 |

### 介護業界の現実

- 介護職の有効求人倍率: **約3.5倍**（全業種平均1.2倍の約3倍）
- 年間離職率: **約15%**（身体的負担が主因の一つ）
- 1人当たり採用コスト: **約30〜50万円**（求人広告+紹介手数料）
- 慢性的な人手不足で、**求人を常時出している施設が大半**

### なぜ「求職者」への直接アプローチは効率が悪いか

- 求職者は購買決定者ではない
- 転職検討中の介護士に「YUMEHO」を訴求しても購買行動に繋がらない
- 広告費が分散する（リーチは広いが CV が遠い）

### なぜ「求人を出している施設」へのアプローチは有効か

- 求人を出している ＝ 人手不足 ＝ YUMEHO で解決できる課題を持っている
- 求人コストに年間数百万円かけている ＝ 投資意欲がある
- 「人が採れない → 設備で解決する」という意思決定の文脈が作れる

## 3. 戦略 A — 求人データからの見込み施設特定

### 3.1 コンセプト

求人サイトに掲載されている施設情報を **見込み客リストとして活用** する。  
「常に人を探している施設」＝「人手不足が深刻な施設」＝「YUMEHO の最有力ターゲット」。

### 3.2 ターゲットとなる求人の条件

以下の条件に合致する求人を出している施設を抽出する。

| 条件 | 理由 |
|------|------|
| 職種: 理学療法士（PT）・作業療法士（OT）・介護職員 | YUMEHO の直接利用者 |
| 施設種別: 回復期リハビリ病院、介護老人保健施設、通所リハビリ、デイサービス | YUMEHO の導入実績がある施設種別 |
| 掲載期間: 60日以上 or 繰り返し掲載 | 慢性的な人手不足の証拠 |
| 所在地: まず首都圏・関西圏・東海圏 | 営業効率の高いエリアから |
| 施設規模: 50床以上 | 投資判断ができる規模 |

### 3.3 データソース

| サイト | 特徴 | 活用方法 |
|--------|------|---------|
| Indeed | 最大級の求人集約サイト | 施設名・所在地・職種で検索 |
| ジョブメドレー | 医療介護特化 | PT/OT/介護の求人が豊富 |
| カイゴジョブ | 介護職特化 | 施設種別での絞り込みが容易 |
| ハローワーク求人検索 | 公的データ | 掲載期間が長い求人を抽出可能 |
| e介護転職 | 介護施設特化 | 施設プロフィールが詳細 |

### 3.4 リスト作成手順

**Phase 1: 手動リサーチ（初期100施設）**

1. 上記サイトで「理学療法士 回復期」「介護職員 リハビリ」等で検索
2. 60日以上掲載 or 複数サイトに同時掲載している施設を抽出
3. 施設名・所在地・規模・担当者名（可能なら）をスプレッドシートに記録
4. 施設の公式サイトを確認し、リハビリ方針・設備情報を追記

**Phase 2: 定期モニタリング（月次）**

1. 同条件で月1回検索
2. 「3ヶ月以上継続して求人を出している施設」を最優先リストへ昇格
3. 新規掲載施設を追加

**Phase 3: 自動化（余力があれば）**

- Google Alerts で「理学療法士 求人 回復期リハビリ」等を設定
- Indeed RSS フィードを監視
- スプレッドシートに自動追記する仕組み（GAS 等）

## 4. 戦略 B — 「採用に効く YUMEHO」という訴求軸

### 4.1 コンセプト

YUMEHO の導入メリットを「人件費削減」だけでなく、  
**「採用力強化」**という新しい角度から訴求する。

#### 発想の転換

```
従来の訴求:
  「人を減らせます」（コスト削減）

新しい訴求:
  「人が集まります」（採用力強化）
  「人が辞めません」（定着率向上）
```

### 4.2 なぜ「採用に効く」と言えるか

#### 求職者が施設を選ぶ基準

介護職の転職理由と施設選びの優先順位（各種調査より）:

| 順位 | 転職理由 / 重視点 | YUMEHO との関連 |
|------|-------------------|----------------|
| 1 | 身体的負担が大きい | YUMEHO で介助の身体負担が大幅軽減 |
| 2 | 人間関係・職場環境 | 余裕のある人員配置 → 職場環境改善 |
| 3 | 給与・待遇 | 削減した人件費を既存スタッフの待遇改善に充当可能 |
| 4 | キャリアアップ | 最新設備での経験 → スキルの差別化 |
| 5 | 施設の理念・方針 | 「プレイ型リハビリ」は先進的な理念を示す |

つまり、YUMEHO は求職者が重視する **上位5項目すべてに間接的に貢献** する。

#### 求人広告での差別化

YUMEHO 導入施設が求人広告に書ける訴求:

```
【設備・環境】
・最新の天井型歩行リハビリ支援システム（YUMEHO）完備
・ハーネスによる安全確保で介助の身体負担を大幅軽減
・1名の見守りで安全にリハビリを実施できます
・ボール投げ・バドミントンなどの「プレイ型リハビリ」を実践

【この施設ならではの魅力】
・先端設備による安全な環境で、利用者様の笑顔を引き出せます
・身体介助の負担が少なく、リハビリの専門性に集中できます
```

→ 他施設の求人と **明確に差別化** できる

### 4.3 「採用コスト削減」の試算

施設が YUMEHO 導入で採用活動を減らせる構造:

```
【Before: YUMEHO なし】
  歩行リハビリに常時3名必要
  → 離職率15% → 年間0.45人が退職
  → 補充採用コスト: 0.45 × 40万円 = 18万円/年
  → 求人広告費: 月5万円 × 12ヶ月 = 60万円/年
  → 採用関連コスト合計: 約78万円/年

【After: YUMEHO あり】
  歩行リハビリは見守り1名で運用
  → 必要人数が減る → 採用プレッシャーが減る
  → 身体負担軽減 → 離職率低下
  → 先端設備 → 求人の訴求力向上 → 採用効率向上
  → 採用関連コスト: 推定50%削減 = 約39万円/年

  年間削減効果: 約39万円（採用コスト分のみ）
```

**人件費削減254万円 + 採用コスト削減39万円 = 年間約293万円の総合効果**

この試算を稟議資料に追加できる。
